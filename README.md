# Go Gin App Example Skeleton

An example of gin contains many useful features


## Installation
```
$ go get github.com/baturorkun/go-gin-app
```

## How to run

### Required

- Mysql
- Redis

### Ready

Create a **blog database** and import ( /docs/sql/db.sql )

### Conf

You should modify `conf/app.ini`

```
[database]
Type = mysql
User = root
Password =
Host = 127.0.0.1:3306
Name = blog
TablePrefix = blog_

[redis]
Host = 127.0.0.1:6379
Password =
MaxIdle = 30
MaxActive = 30
IdleTimeout = 200
...
```

### Run
```
$ cd $GOPATH/src/go-gin-app

$ go run main.go 
```

### Docker Run
```
$ docker-compose up

```
